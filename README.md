# 用法

+ 编辑`docker-compose.yml`
    + 修改`TAG` 为指定的 python 版本
    + 修改`SRC_LINK` 为需要 build 的链接

+ (可选) 编辑`build_dist.sh` 添加其他依赖

+ 执行命令 `docker-compose up`

## Usage 1

+ editor `docker-compose.yml`

+ run `docker-compose up`

## Usage 2

`make build`
